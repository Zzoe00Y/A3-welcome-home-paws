# 🐾 Welcome Home, Paws

An interactive data visualization exploring adoption patterns for cats and dogs at the Austin Animal Center in 2024.

## 🌐 Live Demo
https://zzoe00y.github.io/A3-welcome-home-paws/

---

## 📖 Overview
This project tells a data story about how **age, time, and animal type** influence shelter outcomes.

It combines:
- a **monthly adoption trend** (overview)
- an **age vs. outcome chart** (detail)
- interactive filters and pet-level exploration

Together, these views help reveal both **system-wide patterns** and **individual animal stories**.

---

## ✨ Features

- 📅 **Monthly Adoption Trend**
  - Shows how adoption rates change across 2024  
  - Click a month to filter the entire visualization  

- 🐶🐱 **Interactive Scatter Plot**
  - Each pet is represented as an icon  
  - Position shows age and outcome  

- 🎛️ **Dynamic Filters**
  - Animal type (All / Dogs / Cats)  
  - Outcome focus (Adopted / Not Adopted)  
  - Age range slider  

- 🔎 **Detail-on-Demand**
  - Click any pet to view detailed information  

---

## 🧠 Key Insights

- **Age is the strongest factor**: younger pets are much more likely to be adopted  
- **Adoption remains consistently high**, but varies across months  
- **Outcomes diversify with age**, especially for older animals  
- Patterns emerge at scale, but each point represents an individual story
